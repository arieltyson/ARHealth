## Project Description üé®

"ARHealth is an Augmented Reality (AR) Wellness Companion app for VisionOS that promotes healthy habits and mindfulness through interactive AR experiences. The app leverages VisionOS SDK and ARKit to overlay wellness activities and prompts in the user‚Äôs real-world environment. It integrates WebKit to provide web-based wellness content, uses HealthKit to track health metrics, incorporates Messages for social sharing and encouragement, and employs SiriKit for hands-free voice commands."
